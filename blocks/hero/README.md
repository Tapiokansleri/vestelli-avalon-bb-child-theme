# Hero Block

Gutenberg block for Vestelli theme.

## Building the Block

To build the JavaScript for the block editor:

1. Install dependencies:
```bash
npm install
```

2. Build the block:
```bash
npm run build
```

3. For development with watch mode:
```bash
npm run start
```

The build process will create a `build/` directory with the compiled JavaScript and CSS files.

## Block Features

- Title and description (RichText)
- Background image with overlay
- Customizable overlay color and opacity
- Text color customization
- Text alignment (left, center, right)
- Custom height
- Two CTA buttons with customizable text and URLs
- Responsive design
